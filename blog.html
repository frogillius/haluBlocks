This project is a recreation of scratch. I am working on this because I feel like scratch doesn't authentically teach all coding concepts it could.


The aim of the project is to provide a simple interface without restricting developers.



Planned features:

-Ease of use, blocks will be abstracted from normal fucntions.


-Usability, more advanced features will be provided for advanced users.


-Fully custom, users will be able to choose from several project templates, or make it themselves.
